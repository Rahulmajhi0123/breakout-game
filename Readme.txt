Breakout Game in JavaScript

This is a classic Breakout game implemented using HTML, CSS, and JavaScript.

## Game Overview

Breakout is a classic arcade game where the player controls a paddle to bounce a ball and break a wall of bricks. The goal is to clear all the bricks on the screen using the ball and the paddle while preventing the ball from falling off the bottom of the screen.

## How to Play

1. Open the "index.html" file in a web browser to start the game.
2. Move your mouse left and right to control the paddle's movement.
3. Click the "Start" button to begin the game.
4. The ball will bounce off the walls and the paddle. Your goal is to hit the bricks with the ball to break them.
5. If the ball hits the bottom of the screen, you lose a try. You have a limited number of tries.
6. When all bricks are broken, your score increases.
7. You can adjust the game level using the dropdown menu.

## Files and Structure

- `index.html`: The main HTML file that displays the game interface and includes scripts and styles.
- `styles.css`: The stylesheet that defines the game's visual appearance.
- `breakout.js`: The JavaScript file containing the game logic and interactions.
- `red.png`, `blue.png`, `yellow.png`, `green.png`: Images for different colored bricks.
- `paddle.png`, `ball.gif`: Images for the paddle and the ball.

## Game Mechanics

- The paddle follows the mouse movement for control.
- The ball's direction changes when it hits the walls, paddle, or bricks.
- Bricks are broken when the ball collides with them.
- The game ends when the ball hits the bottom of the screen and you run out of tries.
